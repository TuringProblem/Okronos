# What is this?

A ***microservice*** [^1] that needs the following to run:

- ***[Node](https://nodejs.org/en/download) for npm*** (*latest version*)

thats it! Now run the following (*feel free to copy it :P*):

```bash
npm i # This will install the dependencies
npm run build # to build the project
npm start # to run the local server
```



[^1]: In layman terms, a  ***microservice*** is just a program that you run independently from the overall project. So similiar to how you run the ***Server*** and ***client*** - they are seperate microservices interacting.
